📚 Acerca de este Laboratorio Virtual

Este laboratorio virtual ha sido diseñado para estudiantes de física que desean profundizar en la comprensión de conceptos fundamentales mediante simulaciones interactivas. Cada simulador permite manipular variables clave y observar resultados numéricos y gráficos en tiempo real.
