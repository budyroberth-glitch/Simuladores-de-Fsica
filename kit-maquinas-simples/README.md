# 🏗️ Kit de Máquinas Simples - Simuladores Educativos

Colección completa de simuladores interactivos para enseñanza y aprendizaje de máquinas simples en física.

## 📋 Simuladores Incluidos

### 1. **Sistema de Palanca** 🔄
Simulador interactivo del principio de momento y equilibrio.

**Características:**
- Ajuste de pesos en ambos lados (0-500g)
- Control de distancias desde el punto de apoyo (0-37cm)
- Cálculo automático de momentos (g·cm)
- Indicador de equilibrio en tiempo real
- Visualización de la inclinación de la palanca

**Componentes del equipo:**
- Base de hierro fundido (700-900g)
- Varilla cilíndrica de acero inoxidable (25cm alto, 1.2cm diámetro)
- Regla graduada de aluminio (74cm largo)
- Sujetadores de plástico ABS
- Balancín con aguja indicadora
- Pesas de 50g y 100g

### 2. **Sistema de Polea** ⚡
Simulador de poleas con diferentes configuraciones.

**Tipos de poleas:**
- **Polea Fija:** Ventaja mecánica 1:1
- **Polea Móvil:** Ventaja mecánica 2:1
- **Polea Compuesta:** Ventaja mecánica 4:1

**Características:**
- Selección de tipo de polea
- Control de peso de carga (50-1000g)
- Ajuste de fuerza aplicada
- Medición de distancia de cuerda tirada
- Cálculo de ventaja mecánica
- Cálculo de altura levantada
- Indicador de estado del sistema

**Componentes del equipo:**
- Marco de aluminio resistente
- Poleas de plástico reforzado (diámetro 8cm)
- Cuerda de nylon resistente (2mm diámetro, 2 metros)
- Gancho de carga de acero inoxidable (capacidad 1kg)
- Escala en cm para medir desplazamientos
- Pesas de 50g, 100g y 200g

## 🚀 Cómo Usar

1. **Acceder al simulador:**
   - Abre `index.html` en tu navegador
   - No requiere servidor web
   - Compatible con navegadores modernos

2. **Navegar entre simuladores:**
   - Usa las pestañas en la parte superior
   - "Inicio" - Descripción general del kit
   - "Sistema de Palanca" - Simulador de palanca
   - "Sistema de Polea" - Simulador de polea

## 💻 Requisitos Técnicos

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- HTML5 Canvas API
- JavaScript habilitado
- Resolución mínima: 800x600 (optimizado para 1200x600)

## 📊 Funcionalidades

### Palanca
- Visualización en tiempo real del equilibrio
- Cálculo instantáneo de momentos
- Indicador visual de inclinación
- Panel de información detallada
- Botón de reinicio

### Polea
- Visualización de 3 configuraciones diferentes
- Cálculo automático de ventaja mecánica
- Determinación de fuerza necesaria
- Cálculo de altura levantada
- Indicador de estado (balanceado, insuficiente, excesivo)
